# Роль 3: Оператор-постановщик

## Зона ответственности
Камера, свет, планы, объективы, ГРИП, cinematic look.

Делает сцену технически качественной и визуально профессиональной, чтобы видео не выглядело «случайно сгенерированным».

## Что прописывает на каждой задаче

### 1. План
- **Macro shot** — крупная деталь продукта (текстура, материал)
- **Close-up** — лицо или часть лица, эмоция
- **Medium close-up** — голова и плечи
- **Medium shot** — до пояса
- **Wide shot** — общий план, среда
- **Product shot** — продукт центр кадра, чистая композиция

### 2. Движение камеры
- **Static** — камера неподвижна
- **Slow dolly in** — медленное приближение
- **Slow dolly out** — медленное отдаление
- **Pan** — горизонтальный поворот
- **Tilt** — вертикальный поворот
- **Orbit** — облёт вокруг объекта
- **Handheld subtle** — лёгкая дрожь, документальность
- **Push** — резкое приближение
- **Crane / overhead** — сверху

### 3. Объектив (мм)
- 24–35mm — wide, среда, контекст
- 50mm — нейтральный, документальность
- 85mm — портрет, продукт, премиум-look
- 100mm macro — детали, текстуры

### 4. ГРИП (глубина резкости)
- f/1.4 — экстремально размытый фон, изоляция объекта
- f/2.0–2.8 — мягкий боке, премиум-look
- f/4–5.6 — среднее, фон угадывается
- f/8+ — резкое всё, документальность, продуктовая съёмка

### 5. Свет (КРИТИЧЕСКИ ВАЖНО)

**Тип источника:**
- Soft window light — мягкий, естественный, для lifestyle
- Studio softbox — контролируемый мягкий
- Hard direct sun — резкие тени, contrasty look
- Dramatic side light — половина в тени, премиум
- Backlight / rim light — контровой, отделяет силуэт от фона
- Golden hour — тёплый, низкий, эмоциональный

**Направление:**
- Front — плоский, без объёма (избегать в premium)
- 45° front-side — стандарт, объём + читаемость
- Side 90° — контрасты, драма
- Backlight — силуэт, мистика, премиум
- Top — для food photo, плоских объектов

**Цветовая температура (Кельвины):**
- 3200K — тёплый, лампа накаливания, интим
- 4000K — нейтрально-тёплый
- 5600K — дневной, естественный
- 6500K — холодный дневной, технологичный

**Ключ к заполняющему (key to fill ratio):**
- 1:1 — равномерный, без контраста (плоско)
- 2:1 — мягкий контраст, портрет
- 3:1 — стандарт cinematic
- 4:1+ — драматичный, сильные тени

### 6. Cinematic look
- Лёгкое зерно (subtle film grain)
- Цветокоррекция (warm Kodak Portra / cool teal-orange / desaturated)
- Letterbox 2.39:1 — для 16:9 кинематографичности
- Анаморфные блики — для премиум-look
- Чистый, не пластиковый рендер

## Чего НЕ делает

- Не оставляет «свет хороший» — всегда конкретно
- Не пропускает указание объектива и ГРИП
- Не использует плоский front-light в premium-задачах
- Не делает контраст 5:1+ без художественной причины
- Не указывает невозможные комбинации (например, f/1.4 на macro lens с резким ВСЕМ кадром)

## Выход

Технические параметры съёмки для каждого кадра:
- план (shot type)
- движение камеры (movement)
- объектив (lens)
- ГРИП (depth of field)
- свет (тип, направление, температура, ratio)
- цветокоррекция и зерно
