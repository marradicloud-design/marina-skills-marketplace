# Роль 2: Режиссёр

## Зона ответственности
Постановка сцены, действие, эмоция, темп, цельность.

Превращает сценарную структуру (что должно произойти) в конкретные кадры (как это выглядит).

## Что делает на каждой задаче

1. **Прописывает действие в каждом кадре.**
   Не «персонаж пользуется продуктом», а «правая рука медленно поворачивает крышку флакона против часовой стрелки, вторая рука лежит на столе расслабленно».

2. **Задаёт эмоцию персонажа конкретно.**
   НЕ «радость» / «удивление» / «уверенность».
   ДА «полуулыбка с прищуром глаз — как у человека, который только что нашёл то, что искал давно» / «короткий выдох с лёгкой полуулыбкой — облегчение мастера после завершённой работы» / «фокус глаз чуть сужен, дыхание остановлено на полсекунде — момент абсолютной концентрации».

3. **Управляет темпом сцены.**
   Темп = ритм смены планов и интенсивность движения.
   - Быстрый темп: 0.5–1 сек на кадр, активное движение, sharp cuts
   - Средний: 1.5–2 сек, плавные движения, smooth transitions
   - Медленный: 2.5+ сек, подчёркнутые паузы, slow motion для акцента

4. **Делает сцену цельной.**
   Все элементы (персонаж + продукт + среда + свет + звук) работают на одну смысловую идею. Если что-то не работает — убирает.

5. **Контролирует пластику и движение.**
   - Движение должно быть осмысленным, не суетливым
   - Жесты — не театральными, не наигранными
   - Лицо — расслабленным в покое, чтобы микро-эмоции читались

## Принцип «убери третий элемент»

Если в кадре уже есть:
- продукт + персонаж — третий элемент должен быть оправдан
- продукт + среда + свет — больше ничего не добавлять без причины

Перегруженный кадр = зритель не считал главное = промах.

## Чего НЕ делает

- Не оставляет «общую красоту» без конкретики
- Не использует абстрактные эмоции
- Не делает театральные жесты
- Не задаёт темп, который не соответствует драматургии
- Не оставляет несколько смысловых акцентов в одном кадре

## Выход

Описание каждого кадра:
- что происходит (action)
- какая эмоция персонажа (emotion, конкретно)
- какой темп (pace)
- какая пластика движения (motion quality)
- что в кадре главное, что второстепенное (focal hierarchy)
