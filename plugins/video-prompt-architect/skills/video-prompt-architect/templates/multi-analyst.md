# Мульти-аналитика: Claude + Gemini + GPT в пайплайне

Эти модели не первичные генераторы визуала. Они — аналитики, которые усиливают работу основных моделей (Kling, Seedance, Nana, Runway, Midjourney, Nano Banana, GPT Image 2).

## Распределение ролей

| Модель | Сильная сторона | Когда подключать |
|---|---|---|
| **Claude** | Структурный аудит, длинный контекст, JSON-сборка, переписывание под коммерческую цель | Финальная сборка промта, переписывание слабого промта, сценарный аудит, работа с длинными ТЗ |
| **Gemini** | Мультимодальный анализ изображений и видео, нативная сверка кадров | Распаковка референса, сверка консистентности персонажа между кадрами, разбор полученного результата визуально |
| **GPT (ChatGPT, GPT-5/4o)** | Креативный brainstorm, скорость итерации, альтернативные формулировки | Генерация альтернативных идей, sanity-check, быстрые варианты заголовка/CTA, brainstorm креативных ходов |

## Когда что подключать

### Claude →
- Сборка финального JSON-промта под Kling / Seedance / Runway / GPT Image 2
- Переписывание слабого промта пользователя в сильный с сохранением замысла
- Аудит сценария 15-сек ролика по драматургии (хук / удержание / CTA)
- Длинные ТЗ от клиента → структурированный бриф под генерацию
- Адаптация одного промта под несколько моделей (Kling → Seedance, MJ → Nano Banana)
- Работа с большими корпусами референсов и моодбордами

### Gemini →
- Получает картинку-референс → выдаёт структурный промт под Midjourney / Nano Banana / GPT Image 2 для воспроизведения стиля
- Сравнивает 2–4 кадра подряд, ищет дрейф лица, одежды, продукта между кадрами
- Анализ сгенерированного ролика: что не сработало, где промт не выполнен
- Проверка консистентности продукта от стартового кадра к финальному в image-to-video

### GPT →
- 5–10 альтернативных идей промо-ролика за один запрос
- Brainstorm заголовков и CTA в кадре
- Альтернативные креативные ходы, когда основная идея не цепляет
- Быстрая sanity-проверка: «звучит ли это естественно по-русски / по-английски»
- Конкурентные референсы и trending-формулы по категории

## Шаблоны вызова

### Claude — финальная сборка промта

```
Роль: AI-режиссёр и архитектор промтов для коммерческой генерации видео и изображений.

Контекст:
- Целевая модель генерации: [Kling / Seedance 2 / Runway / Midjourney / Nano Banana / GPT Image 2]
- Платформа: [Reels / TikTok / WB карточка / Stories]
- Коммерческая цель: [клик / запоминание / прямая продажа]
- Стиль и тон: [описание]
- Продукт и его ключевая характеристика: [описание]

Задача:
Собери финальный промт под целевую модель, используя 4-ролевой workflow (сценарист → режиссёр → оператор → креативный директор). Выдай в формате JSON-шаблона из этого скилла. Прогони через 13-пунктовый чек-лист. Добавь negative prompt и краткий комментарий.
```

### Gemini — мультимодальный аудит

```
Роль: AI-аналитик визуальной консистентности.

Вход: [прикрепить 2–4 кадра / стартовый кадр + полученный ролик / референс]

Задача:
[выбрать одно]
1. Сравни эти 4 кадра на консистентность персонажа/продукта. Опиши дрейф: что изменилось в лице, одежде, продукте, освещении.
2. Распакуй стиль этого референса в структурный промт под [Midjourney / Nano Banana / GPT Image 2].
3. Промт был [вставить промт]. Полученный кадр [прикрепить]. Где промт не выполнен и почему.

Формат: диагноз → конкретные правки промта → готовая версия.
```

### GPT — креативный brainstorm

```
Роль: креативный директор рекламного агентства, специализация — performance-креативы для маркетплейсов.

Контекст:
- Продукт: [описание]
- Аудитория: [портрет]
- Платформа: [Reels / TikTok / Shorts / WB]
- Длительность: [5 / 15 / 30 сек]

Задача: дай 7 принципиально разных креативных ходов для промо-ролика. Каждый — одно предложение с хуком и payoff. Без эстетики ради эстетики, всё должно продавать.
```

## Когда НЕ подключать аналитиков

- Когда задача узкая и шаблонная (стандартный packshot, типовой lifestyle) — основной workflow с 4 ролями справится без них
- Когда нужно очень быстро — не тратить раунды на промежуточный аудит
- Когда результат уже хороший — не «улучшать» ради улучшения

## Интеграция в основной workflow

```
Вход (бриф / ТЗ / слабый промт пользователя)
       ↓
[опц. GPT — brainstorm 5–7 креативных ходов]
       ↓
Сценарист → Режиссёр → Оператор → Креативный директор
       ↓
[опц. Claude — финальная сборка JSON под конкретную модель]
       ↓
[опц. Gemini — аудит консистентности, если есть стартовый кадр или предыдущий ролик]
       ↓
Финальная отдача пользователю: основной промт + negative prompt + комментарий
```

Аналитики подключаются избирательно. Стандартный workflow без них тоже работает — это усилитель, а не обязательный шаг.
