# Чек-лист качества промтов (13 пунктов)

Применяется перед выдачей ЛЮБОГО промта. Не прошёл хотя бы один пункт → переделать.

## Видео-промты (Veo 3, Seedance 2)

### 1. Цель ролика
Из промта однозначно понятно: продаём / привлекаем внимание / упаковываем / создаём имидж. Если непонятно — переделать.

### 2. Хук
В первые 1–2 секунды — событие, движение или неожиданность. НЕ пустой кадр, НЕ медленное появление логотипа, НЕ «общий план без действия».

### 3. Видимость продукта
Продукт не «где-то в сцене», а В ФОКУСЕ. Минимум 40% времени ролика. Для рекламы — ближе к 60%.

### 4. Движение и развитие сцены
Нет статичных кадров дольше 1.5 секунды без художественного обоснования. Камера или объект — что-то развивается.

### 5. Чистота кадра
Не перегружен объектами, фоном, текстом, эффектами. Один кадр — один смысл. Если хочется добавить третий элемент — убери что-то первое.

### 6. Свет
Прописан конкретно: тип источника, направление, цветовая температура, соотношение ключевого и заполняющего. НЕ «хороший свет», НЕ «приятное освещение».

### 7. Камера
Прописан план (close-up / medium / wide / macro / product shot), движение (dolly / pan / orbit / handheld / static), объектив в мм, ГРИП.

### 8. Эмоция
Конкретная, развёрнутая. НЕ «счастье», а «спокойная гордость после завершённой работы». НЕ «удивление», а «жадный интерес ребёнка перед подарком».

### 9. Коммерческий смысл
Если убрать сцену — продажа упадёт? Если нет — сцена бесполезна. Каждый кадр должен либо удерживать, либо показывать продукт, либо строить желание, либо вести к покупке.

### 10. Консистентность
Персонаж, продукт, цвет, свет, стиль не плывут между кадрами. Если работаешь с референсами — блок consistency_anchors заполнен.

### 11. Формат и модель
9:16 / 1:1 / 16:9 указан. Длительность под возможности модели (Veo 3: до 8 сек, Seedance 2: 5–10 сек комфортно). Платформа учтена.

### 12. Negative prompt
Есть. Минимум: distorted, blurry, low quality, extra limbs, watermark, text artifacts, oversaturated. Под задачу — расширить.

### 13. Packshot + CTA
Для рекламы обязательно: финальный кадр с продуктом, логотипом, текстовым CTA. Длительность 1–2 сек.

## Image-промты (Nano Banana, GPT Image 2)

Для image применяются пункты 1, 3, 5, 6, 7, 8, 9, 10, 11, 12. Дополнительно:

### Композиция
- Правило третей / центрированно / золотое сечение — указано
- Subject position указан (left third / center / right third)
- Negative space указан

### Глубина кадра
- Foreground / subject / background — все три плана прописаны (или явно сказано, что фон чистый)

### Для GPT Image 2 дополнительно
- Текст в кадре прописан с учётом read flow (что читается первым)
- Шрифт указан (категория: sans-serif bold / serif elegant / display modern)
- Размер текста относительно кадра указан
- Расположение текста указано (top / middle / bottom + horizontal alignment)
- Контраст текста с фоном проверен (тёмный текст на светлом / наоборот)

### Для Nano Banana дополнительно
- Если кадр пойдёт в image-to-video — указан consistency_for_video_handoff с описанием якорей
