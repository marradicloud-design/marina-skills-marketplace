# Examples

Эта папка наполняется ПОСЛЕ установки и тестирования агента.

## Что класть сюда

После каждой удачной задачи — сохранять:
- исходный бриф
- финальный промт (JSON)
- ссылку на результат (видео/картинка в Google Drive)
- комментарий: что сработало особенно хорошо

## Структура файла примера

```
example-name.md
---
# [Название кейса]

## Задача
[бриф клиента или личная задача]

## Модель
[Veo 3 / Seedance 2 / Nano Banana / GPT Image 2]

## Ключевые решения
[2-3 пункта о том, что главное было заложено]

## Финальный промт
\`\`\`json
{...}
\`\`\`

## Negative prompt
\`\`\`
...
\`\`\`

## Результат
[ссылка на Google Drive]

## Что сработало
[коммерческий эффект — CTR, удержание, отзыв клиента]
```
